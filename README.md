# Tools Factory

Live site Link:  [https://tools-factory.web.app/](https://tools-factory.web.app/).

## `Features`
* Api to get all tools
* Api to delete tools
* Apt to manage order status
* Api to add product
* Api to delete products
* Api to get specific order
* Api to get user specific orders
* JWT veryfication

## `Technology Used`
* Node js
* Express js
* MongoDB
* Cors
* Dotenv
* JSON webtoken
